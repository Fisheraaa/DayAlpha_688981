Central finding: A genuine intraday time-of-day alpha existed in SMIC during 2020–2024, captured by time_cos / time_sin features (22% of total SHAP). By 2025–2026 this regularity had fully decayed — the same model, unchanged, produced a statistically significant loss (p = 0.0001). Pipeline

What Was Built

End-to-end research pipeline from raw tick data to live signal generation, covering all standard production-quality components.

01 · DATA & FEATURES
IC-Based Feature Selection
31 technical / volume / micro-structure features. IC threshold |IC| ≥ 0.008 on rolling 300-bar window. 31 → 28 selected. Baostock 30-min OHLCV, adjusted.
02 · MODELS
4-Model Ensemble
Transformer-LSTM · LSTM · CNN-LSTM · MLP. PurgedKFold cross-validation with gap. Optuna HPO (30 trials/window). Equal-weight ensemble (val-AUC normalised).
03 · EXPLAINABILITY
Three-Layer Attribution
SHAP GradientExplainer + temporal heatmap. Integrated Gradients (n=120). Permutation Importance (n_repeats=3). Feature stability across time windows.
04 · BACKTEST
Dual-Engine Validation
Custom engine: dynamic position sizing (20–40%, confidence-scaled), 3-layer stop-loss (fixed 1.2% · trailing · 3-bar time stop), EOD forced exit. Cross-validated with VeighNa BacktestingEngine. Round-trip cost ≈ 20 bp.
05 · LIVE
TDX Real-Time Feed
pytdx connector with heartbeat reconnect + fallback servers. Live bar buffer → feature pipeline → ensemble inference → signal log. Signal output to CSV every 30-min bar.
06 · VALIDATION
Rigorous OOS Design
11-window Walk-Forward (train 8 mo / test 1 mo / step 1 mo). One independent "double-blind" OOS on 2025-01 → 2026-06 data never seen during any tuning decision. Statistical significance via t-test (Lo 2002).

Statistics

Statistical Significance

Two complementary methods. Method A tests whether individual trade returns are profitable on average. Method B (Lo 2002) tests whether the full equity curve Sharpe differs from zero.

PeriodMethod NMean / Sharpe t-statp-value Verdict
2024 OOS A · per-trade t-test 250 mean = +0.263% +2.4920.0134 ✓ Significant (p<0.05)
2024 OOS B · bar Sharpe t-test 1926 SR = +0.804 +0.7860.4319 ✗ Not significant
2025–26 OOS A · per-trade t-test 358 mean = −0.062% −0.9730.3313 ✗ Not significant
2025–26 OOS B · bar Sharpe t-test 2735 SR = −3.332 −3.8810.0001 ✓ Highly significant (p<0.001)

Method A: H₀: mean per-trade return = 0. Method B (Lo 2002): H₀: annualised Sharpe = 0, t ≈ SR_bar × √N. The positive result (+0.804) was never statistically significant by Method B (p=0.43). The negative result (−3.332) is highly significant (p=0.0001). This asymmetry is consistent with the hypothesis that +4.94% reflects a sampling tail rather than the true expected value.

Ablation Study

Diagnosing the Alpha Decay

If time_cos / time_sin were spurious correlations, removing them should improve in-sample Walk-Forward. If they captured genuine historical regularities, removing them should hurt. Result: removing them hurts — confirming genuine in-sample alpha that subsequently decayed.

Baseline vs Ablation — All Walk-Forward Metrics
Every metric deteriorated after removing the time features (28→26). This rules out "spurious correlation" and confirms "alpha decay".
Logic chain:
Permutation importance (2025–26) says time_sin is harmful → could be spurious correlation that never worked.
Ablation (2024 Walk-Forward) says removing it makes things worse → it was genuinely useful in training era.
Conclusion: NOT spurious correlation. Alpha existed and then decayed.

Why 2025–26? SMIC's price range shifted from ~40–95 yuan (2020–2024) to ~93–130 yuan (2025–2026), moving outside the training distribution. Intraday auction dynamics, institutional participation patterns, and momentum/reversion regimes all likely shifted with the price level and broader microstructure changes in the STAR Market from 2024 onwards.
Key Findings

What This Research Demonstrates

The value of this project is not that it found a profitable alpha. It is that it built the machinery to find, verify, test, and diagnose an alpha — and applied that machinery honestly.

01
A genuine time-of-day alpha existed in SMIC 2020–2024. Three independent lines of evidence: (a) positive Walk-Forward Sharpe windows existed; (b) permutation importance ranked time_sin as #1 most useful feature; (c) ablation experiment confirmed removal degrades Walk-Forward performance.
02
That alpha was not durable beyond the training era. On a fully independent OOS (2025-01 to 2026-06-12 — never touched during any tuning decision), the same model produced Sharpe −3.332, statistically significant at p = 0.0001. The positive 2024 result (+0.804, p = 0.43) was never statistically significant.
03
The decay mechanism was identified through feature attribution. SHAP remained stable (same top-5, same 22% share) — the model continued to rely on time features. Permutation importance showed time_sin flipped from rank #1 (useful) to rank #27 (harmful). This means: the intraday timing regularity changed, but the frozen model kept betting on the old pattern.
04
Repeated trials on 2024 inflated the reported result. ~16 configuration decisions were made by observing 2024 OOS outcomes, creating selection bias analogous to López de Prado's "backtest overfitting via repeated trials." The Walk-Forward distribution (mean Sharpe −2.42) is a better estimate of true expected value than the selected single point (+0.804).
05
Next iteration: expand training, study microstructure regime shifts. Retraining on 2024–2025 data would allow the model to learn current time-of-day dynamics. Cross-sectional expansion (peer STAR Market names) would improve statistical power (250 trades → 250×N). Adding a regime-detection layer to detect when time-of-day signal quality deteriorates would provide a natural risk-off mechanism.
